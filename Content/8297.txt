position name: data scientist 15-0904-01_cho/dc data scientist                                                 job location: charlottesville, va, or arlington, va start date: asap number of openings: multiple education:  bachelor's degree or advanced degree in data science, mathematics, computer science, statistics, operations research, engineering, or 5+ years of equivalent experience.   years of experience: 5+   desired skills include:  

proficiency with different programming tools (r, sas, python, c++, and/or idl)
proficient programming skills to extract, transform and clean multi-terabyte data streams
data analysis skills including but not limited to multi-variate regression, cluster analysis, principal component analysis
experience with big data processing (i.e. mapreduce, hadoop) as well as databases (amazon aws, mongodb, cassandra)
previous experience with web scraping or social media data analytics preferred
excellent communication and interpersonal skills and experience in communicating and presenting results to a broader group

  primary responsibilities: the data scientist will assist scientific and technical staff in the interpretation, integration, transformation, reporting of large data sets ("big data") from multiple scientific background sources. the individual will be able to process and deconstruct information from multiple different data types and data sources to support assigned missions. individual should be able to exploit and parse data using a wide set of tools (i.e. arcgis, gale, envi, stk). the data scientist will work in a team environment with support from technical staff and will mostly communicate with scientific staff.   clearance status: this position requires that the candidate be willing and able to complete a successful background screening for a security clearance. candidates with an active security clearance will receive preference.   citizenship status: u.s. citizen
