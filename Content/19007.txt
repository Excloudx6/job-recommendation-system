web application developer opportunity description


web application developer 


research & development
full time



i'm interested in this position with steadyrain!

the web application developer is responsible for the creation and maintenance of various .net web application projects. the right candidate for this position will implement requirements using various technologies across the full stack, test and debug issues, and use version control software by utilizing knowledge of standard concepts, practices and procedures in application and database development. additional responsibilities include integrating front-end elements built by our front-end developers into applications.   
key accountabilities: 

integration of user-facing elements developed by      front-end developers with server side logic.
building reusable code and libraries for future use.
implementation of requirements and data protection.

knowledge & experience candidate must have to qualify: 

minimum of 3 years' experience with web development in      the following areas.
knowledge of a web development environment including      experience in the following: 

custom development in .net (with c#), asp.net
restful webapi / ajax
javascript / jquery
webforms, mvc, html5, and css3
entity framework and how to write complex linq       statements

proven problem-solving abilities, detail oriented and focus on quality assurance of work

set yourself apart - nice to have: 

dotnetnuke
bootstrap integration
ms sql server, writing complex, optimized queries and stored procedures
knockout.js or comparable data binding jquery library
github
