job description : 
lockheed martin is seeking an information systems analyst with knowledge in microsoft dynamics. candidate must be detail-oriented, have strong analytical skills and the capacity to handle multiple projects and roles. position will support the development, on-going optimization and maintenance of multiple information systems. position will require working directly with user groups and customers to define and clearly document requirements for new system implementations and system enhancement requests. selected candidate will work directly with software developers to develop and test systems implementations and enhancement requests in preparation for production deployments. position provides ongoing direct user support across large and diverse user groups with widely varying technical skill sets. interface with user groups and clients to conduct systems training and systems development consultation.responsibilities include but are not limited to the following:• develop system development technical requirements, systems deployment project management, data analysis and management, user/customer training and on-going systems support.• develop product deployment schedules and work with management to deploy new systems and enhancements (including coordinating downtime and migration)• develop product roadmaps. work with user groups and management teams to develop product enhancement / software improvements and oversee scheduling and execution of enhancements.• maintain library of system enhancements and version controls• ensure system integration with program requirements and conduct user acceptance testing• develop qa/qc plans for system deployments• develop training materials and workshops for multiple user groups.• work across functional groups with wide ranging technical skill setsbasic qualificationsexperience in the following:• it project management • it system requirements development• user support• client communication• microsoft dynamics or similar crm platformdesired skills• it project management and information systems implementation experience. • experience managing it projects within tight deadlines and clearly defined budgets• ability to translate end user requirements into technical specifications • ability to work between user/customer groups and technical software development personnel. • prior software management and implementation experience. • ability to interact regularly with large number of users and provide support and troubleshooting• strong data analytical skills and desire to work with data. ability to analyze large amounts of data, identify discrepancies and implement solutions• knowledge of it system architecture and cloud-based hosting arrangements is a plus• html and .net experience a plus. • adept at utilizing microsoft office applications (excel, access, power point), ms dynamics crm, share point, sql server, scribe and others. • understanding of crm systems – such as salesforce a plus.• ability to lead cross functional teams, coordinate activities across disparate groups, manage ambiguity, prioritize, and multi-task will be essential• experience in strategic and business plan development, including implementation of vision and strategy• public speaking/presentation experience a plus• demonstrated experience customizing crm systems both through the application and using custom java script• willingness and ability to influence team members outside direct reporting structure• ability to analyze and interpret technical documentation and provide descriptions of technical features to non-technical audiences• experience creating client-ready reports, charts and data analysis using microsoft excel• experience procuring and managing software licenses for third-party
