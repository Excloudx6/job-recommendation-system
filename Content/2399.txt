job description : 
plan and conduct wind tunnel testing including, instrumentation set-up and data acquisition programming for obtaining data during test under the direction of the lead test engineer and wind tunnel manager. create data reduction software for reducing raw data into usable engineering format per customer supplied data requirements. candidate must be able to perform computer programming, set-up and modification of computer programs for wind tunnel test systems control and data acquisition, processing and software maintenance as well as new systems development. position entails remaining current in the latest wind tunnel testing techniques including support of instrumentation calibration software maintenance and upgrades. candidate must possess the initiative to continuously improve tools and processes for ever increasing test productivity. ability to program in a pc and hp mainframe environment is required. must be able to work in a team environment with other wind tunnel personnel and customer representatives.basic qualificationscandidate must have a basic understanding of aerodynamic principles. shift work is required during active testing. previous experience in a test environment is desirable. experience with fortran, visual basic and labview is desirable. must possess or be able to obtain a dod secret level security clearance. degree in a related engineering or computer programming discipline is required.desired skillsfortran visual basic matlab labviewtest experiencewind tunnel testinghp 1000as a leading technology innovation company, lockheed martin’s team
