overview






this .net web application software engineer position will support the air force human resource (hr) systems division (aflcmc/hih) located at randolph afb, tx, as this position will be focused on supporting and developing web applications, including afmilpers, vmpf, afpc secure, and other web applications.  the person that fills this position will become an expert on the hr systems .net web applications and its subsystems. this role will require experience with .net web applications and interfaces with other application systems (including c#, asp.net, mvc, sql server, java and javascript, and oracle products).








qualifications






qualifications: bachelor's degree in a technical related discipline and 5 years of work experience.must have experience with .net framework web application stack.basic understanding of development languages (c#, asp.net, mvc, sql server, java, and javascript, and oracle products).experience with adobe acrobat, power shell and visual studio is desired.have the ability to communicate both orally and in writing with clarity and accuracymust possess mastery of microsoft office productsmust be able to keep up with changing priorities in a fast paced environment.must be a us citizencandiate must have an active security clearance. 








responsibilities






responsibilities:analyze business requirements, designs and writes technical specifications to design or redesign computer solutions.develop original and/or complex code or provides coding guidance to less experienced staff.leads and/or performs modeling, simulations and analysis efforts.verify program logic by overseeing the preparation of test data, testing and debugging of programs.participate in overall systems testing.participate in the migration of applications to production.contribute to project and technical documentation that conforms to mil-std-498.act as an escalation point for application support and troubleshooting, provides guidance and direction to less experienced staff in resolution of escalated issues and/or complex production, application or system problems.may lead work efforts, including assignment of tasks, and provides direction and guidance to less experienced staff.assure quality, security and compliance requirements are met for supported area.participates in the updating and testing of the business continuation plan.provide guidance to implement agile methodologies and devops including automated test and deployment tools.
