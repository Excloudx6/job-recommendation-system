automated test engineer


philadelphia, pa                    

full time                

mid level                


















overview:
we’re looking for an automated test engineer to join our rapidly growing innovation team.
as an automated test engineer, you will be exposed to all facets of the business and be encouraged to learn and develop skills both within and beyond your role. you’ll be working in one of the most collaborative offices in center city philly, developing and working with machine learning technology, surrounded by passionate, high-performing  individuals.
qualifications:
the automated test engineer candidate is an intelligent, motivated person that enjoys designing, building, testing and deploying effective test automation solutions. the ideal candidate has professional experience in the following core areas:

automated test framework development using python, java, or something similar
familiarity with cloud application technologies like docker, kubernetes, and amazon web services
automated test execution as a step in a continuous integration workflow
testing modern web application uis or apis
collaborating with application developers and product managers to improve product quality

responsibilities:

automated test design and development

analyze product features and identify test scenarios in collaboration with developers and product managers
design and develop automated tests using git
peer review automated tests developed by other engineers



automated test framework improvement and enhancement


​​design, develop, and enhance reusable modules that can be used in automated tests


iteratively improve existing test frameworks by collaborating with other developers and automated test engineers




automated test execution analysis​


run automated tests in our continuous integration system,travisci


proactively analyze test results and follow up on test failures


collaborate on improving automated test execution performance 




agile development project participant


participate in agile project planning activities


proactively identify tasks to own and provide regular updates


work with development and product management to meet key project milestones




who we are:
sidecar built a technology from the ground up that’s dedicated to one mission: tackle the challenges retailers face in data-driven e-commerce channels. based on advanced machine learning algorithms and deep data science expertise, our technology digests and analyzes massive, real-time datasets, allowing it to spot trends and continually improve campaigns for google shopping, facebook dynamic ads, and bing shopping.
these channels are changing every day—and have become too complex and competitive to manage manually or with outdated technologies. they require automated, sophisticated, and ever-evolving technology to stay ahead.
