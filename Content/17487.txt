web developer
job description

as a web developer at canviz you will work with the canviz team to develop, deploy, and maintain web sites and web applications. you will work with clients, business analysts, and architects throughout the development lifecycle to help brainstorm web site features, implement web sites and supporting apis, create technical deployment documentation, and assist in deployments with operations personnel.

job skills


3+ years experience working as a web developer
ba in computer science or information systems or equivalent required​​​
experience in microsoft web application development with .net, including asp.net, c#, powershell, html, javascript, css
experience branding web sites to match corporate style guidelines
experience with windows server, iis, sql server, visual studio, tfs
ability to troubleshoot and problem-solve technical and non-technical issues

                                        excellent verbal and written communication skills​​



job duties


develop, deploy, and maintain web sites and web applications. 
collaborate with clients, business analysts, and architects throughout the development lifecycle
help brainstorm web site features, 
implement web sites and supporting apis, 
create technical deployment documentation
assist in deployments with operations personnel.​
