web developer
11.3.2017




the role:

as a web developer, your role will be working on our largest client user-facing sites, mobile applications, social media applications and third-party platforms.  the pace is fast and demanding, but you will be involved with technology solutions that are new and on the cutting-edge.  you will be working closely with a mentor on developing new sites, enhancing existing sites, and learning new technologies.


the responsibilities:
writing client-side code to create dynamic webpages cms driven webpages
working closely with other developers in a team environment
building web-based interactive applications using client based script and hybrid mobile technologies
designing, building, and testing software
taking part in informal reviews of your own and other developer’s code
finding ways to solve difficult problems using initiative
contributing wherever necessary to help the team reach its goals
 
the qualifications:
1-2 years of relevant work experience
client-side programming experience in building cms driven websites 
server-side programming experience of how the browser interacts with the server
understanding of how elements in the dom interact with each other
knowledge of bootstrap css framework and jquery js framework
attention to detail in website development
imaginative and articulate with excellent written communication skills
creativity when solving problems and using new technologies to resolve them
 
the benefits:
salary is negotiable and based on experience
401k with employer match and profit sharing plan
full benefits package, and paid vacations and holidays
software architect and team leadership role advancement available
casual dress code, dog-friendly office
 
the technical: 
html5 usage in modern browsers
css3 animations, structure and parent to child inheritance
.less architecture and dynamic css is a plus
jquery, javascript, and other client scripting packages
familiar with visual studio 2012+, asp.net, c# and microsoft core programming languages.
front-end development using ajax design patterns and interactions with server apis
git code repository management
 
local greater st. louis area candidates only
