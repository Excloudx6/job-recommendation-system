job descriptionui developer 
piper companies is hiring a ui developer for our healthcare it client in the raleigh/durham area. the ui developer will join a team responsible for the development of a state of the art, highly complex and customizable reporting application that aims to improve patient lives and outcomes.
responsibilities of the ui developer include: 

architecting, developing and implementing ui libraries and frameworks that will support a complex, enterprise-level web application.
designing new, highly interactive, user-facing features for the application.
ensuring the ui solutions meet the functionality, scalability and speed of various, diverse clients.

qualifications of the ui developer include:

5+ years of experience in ui/web development in enterprise environments.
extensive experience with javascript, css, and html.
interactive ui development tools such as d3.js, svg or silverlight.
experience with angular js highly preferred, back end experience in c#/.net a plus.

compensation for the ui developer includes:

salary range: $100,000-$120,000
full benefits: cigna healthcare, metlife dental, vsp vision, 401k with american funds

please send qualified resumes to ellen furby at efurby@pipercompanies.com
keywords: ui/ux, ui developer, front-end, web development, full stack, user interface, c#, .net, software engineer, software developer, piper companies, piper enterprise solutions, raleigh, durham, cary, rtp, apex, fuquay-varina, north carolina, nc.
