software developer job description
 
a software developer at donordirect is someone who loves writing code in a large code base and in a dynamic work environment, working in small teams with other developers and product owners and can work with support teams as needed to assist in trouble shooting and resolving issues.  developers may contribute in feature development, new product development, maintenance and defect resolution. 
 
qualifications:
· effective and productive with javascript, css, c# .net and sql
· able to work as a full-stack developer 
· understands object oriented programming
· excellent troubleshooting skills
· effective communicator
· detail oriented
 
responsibilities and expectations:
 
· high quality development and able to write code that is performant and will be easy for others to maintain over time
· complete projects / development on schedule and in scope
· provide development estimates
· proactively communicate impediments and issues
· perform code reviews
· collaborate with members of other teams as needed for troubleshooting or support
· write effective technical documentation when needed
