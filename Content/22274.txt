job description


the advanced technology department at magic leap, inc. is currently in search of a front end engineer to join its team in sunnyvale / san francisco.
the front end engineer will design and develop web applications and visualizations that provide insights into and assist in understanding deep learning models and techniques. you will work on cutting-edge computer vision problems that require unique 2d and 3d visualization solutions. you will work closely with researchers and backend engineers to surface data and results. the tools that you develop will drive key engineering decisions and help guide the work of our deep learning team.
qualified candidates need to be self-starters and able to operate in a highly dynamic environment.
responsibilities:

work closely with deep learning engineers to build next-generation visualization tools and dashboards for analyzing models.
design and implement restful services and front-end systems that provide 2d and 3d visualizations and model performance insights.
be responsible for building tools for specific computer vision problems to inform decisions.
build scalable and highly performant client and server-side web applications.
define application ui/ux for developer tools.

qualifications:

3+ years of software engineering experience.
2+ years of programming experience in server-side languages like python, java, javascript.
provable track record in api design and development
experience with database technologies like mysql, postgresql, and mongodb.
comfortable with web fundamentals like html, javascript, and css
expert in building restful services
familiarity with front-end web frameworks such as angular and react and nodejs.
familiarity working with docker containers.
experience building visualizations using frameworks such as d3 and three.js.
experience with webgl a plus.
solid understanding of web server technologies like node.js, apache, nginx, etc.
experience with deep learning frameworks such as tensorflow and pytorch, or data backend frameworks like spark are a big plus.
