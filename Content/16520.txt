web application developer


virtucrypt is seeking talented individuals with a passion for developing innovative, web application solutions for the role of web application developer. our world-class web application development team is based at futurex's engineering campus in bulverde, texas.this role offers an opportunity for experienced candidates to expand their talents by participating directly in the design, development, deployment, and maintenance of customer-facing web applications. individual visibility within the team is high, and opportunities for rapid advancement exist. all levels of virtucrypt developers work directly on production-level code in a fast-paced, results-oriented environment on products with a rapid time to market.qualified candidates possess the ability to design web applications based on defined requirements; to prepare program specifications; to code, test, and debug software; to interpret technical information relating to web development; to implement intuitive user interfaces and to communicate effectively. the position involves developing and extending both existing and new applications, as well as documenting methods and procedures. primary responsibilities
 design, develop, deploy, and maintain customer-facing web applications  implement intuitive front-end user experiences confer with team members to develop, understand, and update requirements for product features  write and maintain documentation on subjects including code and api functionality maintain confidentiality with regard to information being processed, stored, or accessed benefits
 health, dental, vision, life, and short/long-term disability insurance paid vacation, holidays, and sick leave competitive compensation and opportunities for advancement retirement plan with employer contribution match welcoming, family-style corporate culture uniquely suited to fast-paced, entrepreneurial, and motivated individuals qualificationsour core requirement is that you have a passion for cutting-edge web-based application design and development. html, css, and javascript are essential skills, but if you can prove your aptitude for independent learning, possibly through hyperlinks to hobby or open source projects, that would be even better. we understand that as someone with a true passion for web-based technology, you’re a lifelong learner and can handle anything else we send your way.
bachelor’s degree in computer science, information systems, or related field experience with linux experience with javascript and javascript frameworks experience with version control software (git/svn) strongly prefer
familiarity with python web application developmentexperience with server-client architecturesexperience with rest api developmentfamiliarity with vue.js, react, or angularjs location
futurex's engineering campus in bulverde, texas.
