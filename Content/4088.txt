data scientist - analytics
                        









chicago, il, us
                        








requisition number: 59503 
 
we are looking for an experienced data scientist with practical, hands-on knowledge of advanced analytics architectures and tools such as machine learning, python & r, and the azure data platform. you should be driven to provide quality solutions to challenging problems. you will work closely with client stakeholders and an award-winning team of engineers, architects and thought leaders to design, build, and implement next-generation solutions using microsoft azure.
 
what you’ll do at insight/bluemetal:
 

you’ll build and apply machine learning algorithms and develop models to build enterprise-grade data solutions for a variety of external clients. 
find patterns in manufacturing and financial services data that influence changes in decision making
work directly with client stakeholders to develop technical solutions for business cases. 
aggressively grow your skillset and expertise to meet the emerging needs of our clients. 
	 


what you’ll need to join insight/bluemetal:

2+ years of experience working with data and data analytics development within the microsoft data platform and an excellent grasp of the following technologies:
	
azure machine learning
microsoft machine learning services / r server
using python and r with predictive analytics
		 



2+ years of experience working with probability, statistics, and related techniques in the financial services or manufacturing verticals
	 

exposure to some of the following:
	
microsoft sql server bi stack (ssas, ssis, ssrs)
azure data factory
azure sql database
power bi
		 



strong analytical and reasoning skills that result in clear technical execution.
skill at translating requirements into clean, efficient, quality code
proven ability to prioritize, self-direct and execute at velocity
passion to deliver craftsman-quality work both individually and as part of a team
solid communication skills with both technical and non-technical stakeholders
desire to learn new skills and grow competencies 
bachelor’s degree in computer science or related discipline 
requires travel to chicago area client sites (local only).

the position described above provides a summary of some the job duties required and what it would be like to work at insight. for a comprehensive list of physical demands and work environment for this position, click here.
